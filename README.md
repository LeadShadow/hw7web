Format style in console(server):

    Connection from ('192.168.0.102', 52880)
     received message: hello
    --> hi =)
     received message: how are you?
    --> I'm fine =) and you?
     received message: That's cool me too =)

    --> hello
    received message: hi =)
    --> how are you?
     received message: I'm fine =) and you?
